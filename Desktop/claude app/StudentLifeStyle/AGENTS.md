# Expo

Projektet kører Expo SDK 54 (`expo: ~54.0.0`). Læs de versionerede docs på
https://docs.expo.dev/versions/v54.0.0/ før du skriver kode, og brug altid
`npx expo install <pkg>` så native-moduler matcher SDK 54.

# Projekt-retning

ROADMAP.md i projektroden er den levende plan: nordstjerne, milepæle med checkbokse,
status pr. søjle, forretningsmodel og beslutningslog. Læs den før større ændringer,
afkryds punkter når de er bygget, og tilføj nye beslutninger i beslutningsloggen.
